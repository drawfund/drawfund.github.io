# FileEx-Drawio

#### 简介：

Windows平台上双击drawio文件，可以打开electron，对该drawio文件可以编辑，同时具有自动保存功能。

Logo:

![FileEx-Logo](https://draw.fund/_images/FileEx-Logo.svg)

#### 优点：

- 用户随时可以选择升级electron，且无需再编译。
- 用户随时可以升级-drawio的版本，只需把drawio的webapp文件夹复制进来，替换原有文件夹。
- 可以选择取消electron的菜单栏，会更优美。

#### 零件：

- 当前使用的17.0.0版本的Electron。![Electron_Software_Framework_Logo](https://draw.fund/_images/misc/Electron_Software_Framework_Logo-wh64.svg)
- 当前使用16.6.4版本的Drawio。
